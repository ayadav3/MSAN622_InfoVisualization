Project Sketch

Planned Tools

I plan to use D3.js and R for creating visualizations and preprocessing the data respectively. I also plan to use Tableau for generating the rough sketches of the visualizations.
I plan to use the following javascript libraries:

  • legend.js
  • filter.js
  • biPartite.js

    • Planned Techniques

    I will be implementing 4 techniques on the choosen dataset.

  • Interactive Bar Plot

    • I will use the value or quantity of exports on the y-axis and the continents on the x-axis. As of now, the dataset does not have a column 'Continent'. I will add this column to the dataset soon. Also, I plan to use filters for selecting the commodity and financial year. I will add a check box to sort the bars. I hope to learn concepts of filtering and sorting.

  • Interactive World Map

    • I will use the world map to show the country wise exports of commodities. I plan to use filters to select a commodity and year of exports. The user can select any commodity to see the countrie which imported it. On mouse hover, a tooltip will be displayed to show the name of the countrya and the actual value of export. I hope to learn the use of geo coordinates in D3.

  • Small Multiples (Bar Plot)

    • I plan to use small multiples of bar plots to visualize the change in exports of different commodity categories over the two financial years.

  • BiPartite

    • I plan to use a Bipartite visualization to see the different categories of commodities exported to countries in the two financial year. I will add categories for the commodities to the dataset. I will be using the value of exports in dollar terms for this visualization. Here is the link of a sample BiPartite visualization.

    Planned Interaction

    I plan to use filtering, sorting and panning for making the visualizations interactive. I will use filtering and sorting for the Bar Plots. I will use filters on the World Map visualization. Currently, I am not thinking of linking the interactions between the three visualizations.
